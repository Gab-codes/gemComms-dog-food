# GemCommerce

A premium pet food e-commerce website built with React, TypeScript, and Motion animations for a polished, professional experience.

## Stack

- **React 19.2** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling
- **Motion** - Premium animations

## Getting Started

### Development

```bash
npm run dev
```

Starts the dev server at `http://localhost:5174/`

### Build

```bash
npm run build
```

Produces optimized production build in `dist/`

### Preview

```bash
npm run preview
```

Locally preview the production build

## Features

- **Premium Animations** - Smooth, spring-based motion throughout
- **Responsive Design** - Mobile-first approach with Tailwind
- **Modern Stack** - React 19 with TypeScript for reliability
- **Performance** - Vite for fast builds, optimized animations
